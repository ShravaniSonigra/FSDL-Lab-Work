# HTML Text Formatting Tags

## Overview

This project is a static web page created using HTML5 that demonstrates the commonly used HTML text formatting tags. The tags are presented in a tabular format with their names, descriptions, and example outputs, making it a useful reference for understanding HTML text formatting.

## Objective

The objective of this project is to:

* Understand HTML text formatting tags.
* Learn how to create and organize tables in HTML.
* Display formatted text using different HTML elements.
* Build a simple static web page without using CSS or JavaScript.

## Technologies Used

* HTML5
* Visual Studio Code

## Project Structure

```text
HTML-Text-Formatting/
│── index.html
└── README.md
```

## How to Run the Project

1. Download or clone the project repository.
2. Open the project folder in Visual Studio Code.
3. Open the `index.html` file.
4. Right-click inside the file and select **Open with Live Server** (if the Live Server extension is installed), or open the file directly in any web browser.

## Expected Output

The web page displays a table containing:

* HTML formatting tag
* Description of the tag
* Example output for each tag

## Learning Outcomes

After completing this project, you will be able to:

* Use various HTML text formatting tags.
* Create tables in HTML.
* Differentiate between semantic and non-semantic formatting elements.
* Develop a basic static web page using HTML5.
