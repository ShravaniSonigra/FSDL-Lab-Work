# Online Appointment Booking Application

## Overview

This project is an online appointment booking application developed using Node.js, Express.js, and MongoDB. It allows users to book appointments, view available slots, and manage bookings. MongoDB is used to store user and appointment information.

## Objective

* Develop a web application using Node.js and Express.
* Implement appointment booking functionality.
* Store and retrieve data using MongoDB.
* Perform basic CRUD operations.

## Technologies Used

* Node.js
* Express.js
* MongoDB
* HTML5
* CSS3
* JavaScript
* Visual Studio Code

## Features

* User registration and login (optional)
* Book an appointment
* MongoDB database integration
* Express routing and CRUD operations

## How to Run

1. Open the project in Visual Studio Code.
2. Install dependencies.

```bash
npm install
```

3. Start the MongoDB server.
4. Run the application.

```bash
npm start
```

5. Open the browser and visit:

```text
http://localhost:3000
```
