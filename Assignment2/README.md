# Assignment 2 - Portfolio Website

## Overview

This project is a responsive personal portfolio website developed as part of Assignment 2. It serves as a digital portfolio showcasing personal information, technical skills, academic projects, and contact details. The website is designed using Bootstrap 5 and custom CSS to provide a clean, modern, and responsive user interface.

## Project Structure

```text
Assignment-2/
│── Portfolio.html
│── styleportfolio.css
│── Profile.jpeg
│── Output/
│   ├── output1.png
│   ├── output2.png
│   └── ...
└── README.md
```

## Features

* Responsive navigation bar
* Home section with profile information
* Projects section highlighting academic and personal projects
* Skills section with technical skill icons
* Contact section with email, phone number, and social media links
* Modern UI using Bootstrap cards, gradients, shadows, and hover effects
* Fully responsive layout for different screen sizes

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons
* Google Fonts

## How to Run

1. Download or clone the project repository.
2. Ensure the following files are in the same project directory:

   * `Portfolio.html`
   * `styleportfolio.css`
   * `Profile.jpeg`
3. Open `Portfolio.html` in any modern web browser.
