# Travel Agency Website

## Overview

This project is a simple travel agency website developed using Node.js, Express.js, and MongoDB. It allows users to browse travel packages, view package details, and submit booking inquiries. The application demonstrates basic backend development with NoSQL database integration.

## Objective

* Build a basic travel agency web application.
* Learn server-side development using Node.js and Express.js.
* Store and retrieve travel package data using MongoDB.
* Understand routing and database connectivity.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Node.js
* Express.js
* MongoDB
* Visual Studio Code

## Features

* Home page with travel packages
* View package details
* Booking inquiry form
* Store package or booking data in MongoDB
* Responsive and user-friendly interface

## How to Run

1. Open the project folder in Visual Studio Code.
2. Install the required dependencies:

```bash
npm install
```

3. Start the application:

```bash
node start
```

4. Open your browser and visit:

```text
http://localhost:3000
```

## Learning Outcomes

* Understand Express.js routing.
* Connect a Node.js application with MongoDB.
* Perform basic CRUD operations.
