# Weather Dashboard

## Overview

This project is a simple weather dashboard developed using HTML, CSS, and JavaScript. It visualizes weather-related data such as temperature, humidity, and rainfall using interactive charts created with the Chart.js library.

## Objective

* Create an interactive weather dashboard.
* Visualize weather data using charts and graphs.
* Learn to integrate Chart.js with HTML and JavaScript.
* Present data in a simple and user-friendly interface.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js
* Visual Studio Code

## Project Structure

```text
Weather-Dashboard/
│── index.html
│── style.css
│── script.js
└── README.md
```

## Features

* Displays weather statistics.
* Interactive charts using Chart.js.
* Simple and responsive dashboard layout.
* Easy to understand and navigate.

## Charts Included

* Line Chart – Temperature Trend
* Bar Chart – Humidity Levels
* Pie Chart – Weather Condition Distribution (Optional)

## How to Run

1. Open the project folder in Visual Studio Code.
2. Open `index.html`.
3. Run the project using Live Server or open the file in any web browser.

## Learning Outcomes

* Understand data visualization concepts.
* Learn to use the Chart.js library.
* Create interactive charts using JavaScript.
* Design a basic dashboard interface.
