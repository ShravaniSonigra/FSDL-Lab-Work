# Student Feedback Review System

## Overview

This project is a student feedback review system developed using React.js for the frontend, Express.js for the backend, and MongoDB as the database. It enables students to submit feedback, while administrators can view and manage the submitted reviews.

## Objective

* Develop a full-stack web application.
* Create a responsive frontend using React.js.
* Build REST APIs using Express.js.
* Store feedback data in MongoDB.
* Perform CRUD operations on feedback records.

## Technologies Used

## How to Run

1. Open the project in Visual Studio Code.
2. Install dependencies for both client and server.

```bash
npm install
```

3. Start the MongoDB server.

4. Run the backend.

```bash
cd server
npm start
```

5. Run the frontend.

```bash
cd client
npm start
```

6. Open the browser and visit:

```text
http://localhost:3000
```
